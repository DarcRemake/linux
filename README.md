## Notice
This script is meant to be used on fresh installs of 22.04-20.04 Ubuntu

## How to use:

```
$ wget https://raw.githubusercontent.com/OfficialKimz/linux/main/install.sh
$ sudo chmod a+x install.sh
$ ./install.sh
```
